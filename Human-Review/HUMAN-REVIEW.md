## How this works

- Any conversation that produces something requiring human review saves a file to this folder and adds a row to the index below.
- The **Status** column tracks where things stand: `pending`, `reviewed`, `actioned`, or `archived`.

## Entry format (for Claude)

When adding a file to this folder, append a row to the Index table above using this format:


Files should be named descriptively: `YYYY-MM-DD-topic-slug.md` (e.g. `2026-05-25-github-cli-setup.md`).
